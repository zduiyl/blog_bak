#### Centos7.6 ssh远程不支持中文、中文乱码问题

解决办法：

``` shell
	// 修改配置文件
	vim /etc/locale.conf
	LANG='en_US.UTF-8'
	source /etc/locale.conf
	
	// 查看系统当前语言
	locale
	// 系统支持的语言包
	locale -a
	
	// 修改字体配置
	vim /etc/fonts/fonts.conf
	// 添加中文字体目录
	<div>/usr/share/fonts/chinese</div>
	// 字体存放目录, 将Windows的font下宋体、黑体或者其他字体复制到该目录
	usr/share/fonts/chinese	
	
	// 更新字体缓存
	fc-cache
	// 查看字体列表
	fc-list
	
```

#### ssh远程连接下复制粘贴快捷键
 1. 在终端下：

          复制命令：Ctrl + Shift + C  组合键.

          粘贴命令：Ctrl + Shift + V  组合键.

 

 2. 在控制台下：

          复制命令：Ctrl + Insert  组合键　　或　　用鼠标选中即是复制。

          粘贴命令：Shift + Insert  组合键　　或　　单击鼠标滚轮即为粘贴。
