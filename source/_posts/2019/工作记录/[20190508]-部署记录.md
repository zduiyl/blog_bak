#### 常用命令
开启本地服务
Hexo s --debug

生成静态文件
hexo g //hexo generate

开始部署
hexo d //hexo deploy

#### 使用插件

1.自动摘录
https://github.com/chekun/hexo-excerpt
npm install --save hexo-excerpt

修改你的[主题配置] _config.yml 配置文件如下：

```
	// Deployment
	// Docs: https://hexo.io/docs/deployment.html
	deploy:
	  type: git
	  repo: git@github.com:flyoob/flyoob.github.io.git
	  branch: master
```

2.博客增强插件
https://sulin.me/2019/Z726F8.html
npm install --save hexo-auto-canonical


#### hexo-next部署到github

参考文献：

https://www.cnblogs.com/imapla/p/5533000.html

github Permission denied (publickey) 权限问题
添加ssh-key
https://www.jianshu.com/p/f22d02c7d943

安装github部署插件
npm install hexo-deployer-git --save

完成后部署
$ hexo generate --deploy
$ hexo deploy --generate
